Ensures that the platform instance was constructed with a non-const token that matches the provided token and throws AssertionError if not.

This is used to ensure that implementers are using extends rather than implements.

Subclasses of MockPlatformInterfaceMixin are assumed to be valid in debug builds.

This is implemented as a static method so that it cannot be overridden with noSuchMethod.

@override
Future<void> verify(String domain, String certificate) async {

    'domain': domain,
    'certificate': certificate,
  });
}
